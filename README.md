## 🖥️ System Features

### 1. Interactive Terminal V1 Easter Egg
Accessible via the **Terminal_V1** link in the footer. Launch a fully simulated terminal directly in your browser:
* **Interactive CLI Commands**: Supported inputs include `help`, `whoami`, `ls projects`, `clear`, and `exit`.
* **Command History**: Navigate your inputs using the `ArrowUp` and `ArrowDown` keys.
* **Realistic UX**: Smooth text measuring and rendering, plus a custom blinking block cursor that pauses blinking dynamically as you type.

### 2. Sleek Glassmorphism Stacking
The "About & Mission" section utilizes frosted glass panels (`backdrop-filter: blur(8px)`) over a dynamic skills matrix. This provides high-contrast, modern UI layering that ensures maximum reading clarity without sacrificing visual flair.

### 3. Viewport-Breakout Tech Marquee
An infinite horizontal scrolling background list of skills running on pure, hardware-accelerated CSS keyframe animations. It automatically breaks out to `100vw` for ultra-wide screen coverage, while scaling opacity to act as a clean background watermark.

### 4. Zero Framework Overhead
Built completely on **Vanilla HTML5, CSS3, and ES6 Javascript Modules**. No Tailwind, React, or jQuery dependencies, resulting in:
* **Near-instantaneous loading times** and minimal page weight.
* Highly structured, modularized code following the **Init Function Pattern** for clean DOM queries and state management.

## 🚀 Running Locally

Since this project uses native **ES6 Javascript Modules (`import`/`export`)**, modern browsers restrict local file loading (`file:///`) due to CORS security policies. 

To run this project locally, you must serve it using a local development server:

### Option A: VS Code Live Server (Recommended)
1. Install the **Live Server** extension in VS Code.
2. Click **Go Live** in the bottom right corner of the window.

### Option B: Node.js (via Terminal)
If you have Node.js installed, open your shell in the project folder and run:
```bash
npx serve .
```
